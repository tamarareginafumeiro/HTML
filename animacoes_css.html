O que são animações CSS?

Você pode alterar quantas propriedades CSS quiser, quantas vezes quiser.

Para usar animações, você deve primeiro especificar alguns quadros-chave para a animação.

Os quadros-chave contêm quais estilos o elemento terá em determinados momentos.

A regra @keyframes

Quando você especifica estilos CSS dentro da @keyfranmes regra, a animação mudará gradualmente de estilo atual para o novo estilo determinados momentos.

Para que uma animaçção funcione, você deve vincula a um elemento

Execute a animção na direçãoreversa ou em ciclos alternativos

A animation-derectionpropriedade especifica se a animação deve ser reproduzida para frente, para trás ou em ciclos alterados

A propriedade Animation-Derection pode ter os seguintes valores:

  • Normal- A animação é reproduzida normalmente (avança). Este é o padrão
  • Reversere- A animação é reproduzida na direção reversa (para trás)
  • alternate- A animação é reproduzida primeiro para frente e depois para trás
  • alternate-reverse- A animação é reproduzida primeiro para frente e depois para trás
